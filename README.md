# snAIke

snAIke is an AI generated snake game in python. It works as usual snake game and can be run from command line.
It supports running on both MacOS and Windows machines.

# How it works?

You start as a small snake, travel around the world and eat frogs and birds and mice. As you eat, you grow longer, if you don't eat you become smaller.

# Controls

wasd controls and direction keys.
